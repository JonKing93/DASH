The doc_source subfolder holds the resources needed to build the documentation HTML pages for DASH.

examples: The examples for reference pages, organized in the same file hierarchy as dash.
reST parsers: Functions that convert help text and examples into .rst files for use with Sphinx
sphinx-source-rst: The location of parsed/saved .rst files
sphinx-config: The sphinx configuration file
python-package-mods: Modifications to python packages used to customize the sphinx html pages
    rtd-theme: A modified read-the-docs theme.  Python/Lib/site-packages/sphinx-rtd-theme/state/css/theme.css
    pygments: Highlighting for matlab. Python/Lib/site-packages/pygments/styles/matlab.py

templates: Examples used for testing the Read-the-docs theme
