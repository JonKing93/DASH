PSM.prysm.cellulose.estimate

Estimates cellulose d18O values from T, P, RH, d18Os, and d18Ov

Syntax



d18O = obj.estimate(X)

Description



d18O = obj.estimate(X)

Runs the PRYSM cellulose sensor module on a set of temperatures, precipitations, relative humidities, d18Os, and d18Ov values extracted from a state vector ensemble. Estimates d18O values of cellulose.

Input Arguments

X

numeric array [5 x nMembers x nEvolving]
The temperature, precipitation, relative humidity, d18Os, and d18Ov inputs used to run the PRYSM cellulose sensor module. The first row is temperature, second row is precipitation, third row is relative humidity, fourth row is d18Os, and fifth is d18Ov.

Output Arguments

d18O

numeric matrix [1 x nMembers x nEvolving]
Cellulose d18O estimates produced using the PRYSM cellulose sensor module.
