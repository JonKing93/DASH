dash.math

Functions that implement mathematical operations

Equations

haversine - Calculates distances between latitude-longitude coordinates
gaspariCohn - Implements a 5th order Gaspari-Cohn polynomial
unbias - Computes the coefficient for an unbiased estimator

Ensembles

decompose - Splits an ensemble into its mean and deviations
covariance - Computes the cross covariance over two sets of ensemble deviations
variance - Computes the variance across a set of ensemble deviations

Tests

tests - Unit tests for the dash.math package
