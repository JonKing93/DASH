gridfile.sourcesForLoad

Return the indices of data sources needed to load requested data

Syntax



s = obj.sourcesForLoad(loadIndices)

Description



s = obj.sourcesForLoad(loadIndices)

Returns the indices of the gridfile data sources that are required to implement a load operation.

Input Arguments

loadIndices

cell vector [nDims] {linear indices}
The indices of data elements that are requested for a load operation. Should have one element per dimension in the gridfile. The order of dimensions should match the order of dimensions in the gridfile.

Output Arguments

s

vector, linear indices
The indices of data sources that are required in order to load the requested data.
