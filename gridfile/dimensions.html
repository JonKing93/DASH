gridfile.dimensions

Return the dimensions in gridfile and their sizes

Syntax



dimensions = obj.dimensions


[dimensions, sizes] = obj.dimensions

Description



dimensions = obj.dimensions

Return the list of dimensions in a .grid file.



[dimensions, sizes] = obj.dimensions

Also returns the size of each dimension.

Output Arguments

dimensions

string vector
The dimensions in a gridfile

sizes

vector, positive integers
The size of each dimension in the gridfile.
