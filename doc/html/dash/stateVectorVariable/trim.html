dash.stateVectorVariable.trim

Trim reference indices to only allow complete sequences and means.

Syntax



obj = obj.trim

Description



obj = obj.trim

Remove reference indices that would result in an ensemble member with an incomplete sequence or incomplete mean.

Note: This method can only be called after the variable has been finalized.

Output Arguments

obj

scalar dash.*stateVectorVariable* object
The state vector variable with updated reference indices.
