dash.version

Return the current version of the DASH toolbox

Syntax

dash.version
v = dash.version

Description

dash.version

Prints the current version to the console

v = dash.version

Returns the version as a string

Examples

Print version string to console

dash.version

v4.0.0

Return version string

v = dash.version

v =
   v4.0.0

Output Arguments

v

string scalar
The current version string of the DASH toolbox
