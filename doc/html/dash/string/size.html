dash.string.size

Prints a size as a string of integers separated by xs

Syntax



siz = dash.string.size(siz)

Description



siz = dash.string.size(siz)

Converts a vector of lengths to a string of sizes separated by xs.

Input Arguments

siz

vector, integers
A set of dimension lengths

Output Arguments

siz

char row vector
The size as a string.
