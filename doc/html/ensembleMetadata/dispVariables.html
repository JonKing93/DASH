ensembleMetadata.dispVariables

Prints the variables in a state vector to the console

Syntax



obj.dispVariables

Description



obj.dispVariables

Prints the variables in the state vector to the console. Prints the name of each variable, its length, and the details of its state dimensions.

Prints

Prints a list of variables, lengths, and state dimensions to the console.
