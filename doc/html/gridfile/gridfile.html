gridfile.gridfile

Return a gridfile object for a .grid file

Syntax



obj = gridfile(file)

Description



obj = gridfile(file)

Builds an object for the specified file.

Input Arguments

filename

string scalar
The name of a .grid file.

Output Arguments

obj

gridfile object
A gridfile object for the file.
