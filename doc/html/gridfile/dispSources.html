gridfile.dispSources

List gridfile data sources in the console

Syntax



obj.dispSources


obj.dispSources(objName)

Description



obj.dispSources

Prints a numbered list of gridfile data source files to the console. Links to source details using the name of the object in the calling source.



obj.dispSources(objName)

Specify the name of the object to use when linking to source details.

Input Arguments

objName

string scalar
The name of the object to use when linking to source details.

Prints

Prints a list of gridfile data source files to the console.
