PSM.baymag.baymag

Creates a new BayMAG PSM object

Syntax



obj = PSM.baymag(age, omega, salinity, pH, clean, species)


obj = PSM.baymag(…, options)

Description



obj = PSM.baymag(age, omega, salinity, pH, clean, species)

Initializes a new BayMAG PSM object. Please see the documentation of baymag_forward_ln.m in the BayMAG repository for details about the inputs.



obj = PSM.baymag(…, options)

Provide optional arguments to the BayMAG forward model.

Input Arguments

age

numeric scalar
Age of the observation

omega

numeric scalar
Bottom water saturation state

salinity

numeric scalar
Salinity (psu)

pH

numeric scalar
pH (total scale)

clean

numeric scalar
Describes the cleaning technique

species

string scalar
Name of the target species

options

cell vector
Optional arguments for the BayMAG PSM

Output Arguments

obj

scalar PSM.*baymag* object
The new BayMAG PSM object
