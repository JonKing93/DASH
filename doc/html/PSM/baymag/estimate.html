PSM.baymag.estimate

Estimates Mg/Ca values from sea surface temperatures

Syntax



[MgCa, R] = obj.estimate(SST)

Description



[MgCa, R] = obj.estimate(SST)

Runs the BayMAG forward model on a set of SSTs. Returns an estimate of Mg/Ca values and optionally estimates uncertainties for the estimated values.

Input Arguments

SST

numeric array [1 x nMembers x nEvolving]
Sea surface temperatures used as input for the BayMAG PSM.

Output Arguments

MgCa

numeric matrix [1 x nMembers x nEvolving]
Mg/Ca estimates generated from the SST values using the BayMAG forward model.

R

numeric matrix [1 x nMembers x nEvolving]
Uncertainty estimates for each Mg/Ca value.
