PSM.bayspar.bayspar

Creates a new BaySPAR PSM object

Syntax



obj = PSM.bayspar(latitude, longitude)


obj = PSM.bayspar(latitude, longitude, options)

Description



obj = PSM.bayspar(latitude, longitude)

Initializes a new BaySPAR PSM object. Please see the documentation of TEX_forward.m in the BaySPAR repository for details about the inputs.



obj = PSM.bayspar(latitude, longitude, options)

Provide optional arguments to the BaySPAR forward model.

Input Arguments

latitude

numeric scalar
The latitude of the proxy site in decimal degrees.

longitude

numeric scalar
The longitude of the proxy site in decimal degrees.

options

cell vector
Optional arguments to the BaySPAR forward model

Output Arguments

obj

scalar PSM.*bayspar* object
The new BaySPAR PSM object
