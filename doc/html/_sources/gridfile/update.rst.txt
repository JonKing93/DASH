gridfile.update

Description
-----------

.. raw:: html

    <pre>

.. _gridfile.update.syntax1:

.. rst-class:: syntax

| obj.update

.. raw:: html

    </pre>

Updates the current object to match the content of its .grid file. This ensures that the gridfile object matches the file content, even if the
.grid file is modified by a second gridfile object.



----

Examples
--------

.. rst-class:: collapse-examples

Update a gridfile object
++++++++++++++++++++++++

.. raw:: html

    <section class="accordion"><input type="checkbox" name="collapse" id="example1"><label for="example1"><strong>Update a gridfile object</strong></label><div class="content">


Create two gridfiles pointing to the same gridfile:

::

    grid1 = gridfile('my-gridfile.grid');
    grid2 = gridfile('my-gridfile.grid');


Alter the .grid file. Here, we'll change the fill value.

.. rst-class:: no-margin

::

    fill = grid1.fillValue


.. rst-class:: example-output

::

    fill =
          NaN


.. rst-class:: no-margin

::

    newFill = -999;
    grid1.fillValue(newFill);


The second gridfile object has not been updated, so still reports the old fill value:

.. rst-class:: no-margin

::

    fill2 = grid2.fillValue


.. rst-class:: example-output

::

    fill2 =
           NaN


Update the second gridfile to match the new values in the .grid file:

.. rst-class:: no-margin

::

    grid2.update;
    fill2 = grid2.fillValue


.. rst-class:: example-output

::

    fill2 =
           -999
