gridfile.getLoadIndices
=======================
Organize the dimension indices required to implement a load operation

----

Syntax
------

.. raw:: html

    <pre>

.. rst-class:: syntax

| :ref:`loadIndices = obj.getLoadIndices(userDimOrder, userIndices) <gridfile.getLoadIndices.syntax1>`

.. raw:: html

    </pre>

----

Description
-----------

.. raw:: html

    <pre>

.. _gridfile.getLoadIndices.syntax1:

.. rst-class:: syntax

| :ref:`loadIndices <gridfile.getLoadIndices.output.loadIndices>` = obj.getLoadIndices(:ref:`userDimOrder <gridfile.getLoadIndices.input.userDimOrder>`, :ref:`userIndices <gridfile.getLoadIndices.input.userIndices>`)

.. raw:: html

    </pre>

Takes a user-defined dimension order and accompanying dimension indices and determines the full set of indices needed for a load operation. The full set of indices includes indices for all gridfile dimensions in the order they appear in the gridfile.
