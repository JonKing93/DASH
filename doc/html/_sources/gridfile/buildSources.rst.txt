gridfile.buildSources
=====================
Build dataSources for a gridfile load

----

Syntax
------

.. raw:: html

    <pre>

.. rst-class:: syntax

| :ref:`[dataSources, failed, causes] = obj.buildSources(s) <gridfile.buildSources.syntax1>`

.. raw:: html

    </pre>

----

Description
-----------

.. raw:: html

    <pre>

.. _gridfile.buildSources.syntax1:

.. rst-class:: syntax

| [:ref:`dataSources <gridfile.buildSources.output.dataSources>`, :ref:`failed <gridfile.buildSources.output.failed>`, :ref:`causes <gridfile.buildSources.output.causes>`] = obj.buildSources(:ref:`s <gridfile.buildSources.input.s>`)

.. raw:: html

    </pre>

Builds the dataSource objects for the specified sources. Does not throw errors if a data source fails to build or does not match the gridfile's values for the source. Instead, reports which sources failed and returns the error stack for failed sources.
