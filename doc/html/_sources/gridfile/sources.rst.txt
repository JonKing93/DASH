gridfile.sources
================
Return the ordered list of data sources in a gridfile

Description
-----------

.. raw:: html

    <pre>

.. _gridfile.sources.syntax1:

.. rst-class:: syntax

| :ref:`sources <gridfile.sources.output.sources>` = obj.sources
| sources = obj.sources([])
| :ref:`sources <gridfile.sources.output.sources>` = obj.sources(0)

.. raw:: html

Returns the ordered list of data sources in a gridfile. The index of each data source in the list corresponds to the index of the data source in the gridfile.


.. raw:: html

    <pre>

.. _gridfile.sources.syntax2:

.. rst-class:: syntax

| :ref:`sources <gridfile.sources.output.sources>` = obj.sources(:ref:`s <gridfile.sources.input.s>`)

.. raw:: html

    </pre>

Returns the list of data sources at the specified data source indices. The order of sources in the list corresponds to the order of input indices.
