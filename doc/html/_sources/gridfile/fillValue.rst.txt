gridfile.fillValue
==================
Specify a fill value for data catalogued in a .grid file

Description
-----------

.. raw:: html

    <pre>

.. _gridfile.fillValue.syntax1:

.. rst-class:: syntax

| :ref:`fill <gridfile.fillValue.output.fill>` = obj.fillValue
| :ref:`fill <gridfile.fillValue.output.fill>` = obj.fillValue('default')

.. raw:: html

    </pre>

Return the default fill value for a grid file.


.. raw:: html

    <pre>

.. _gridfile.fillValue.syntax2:

.. rst-class:: syntax

| :ref:`sourceFills <gridfile.fillValue.output.sourceFills>` = obj.fillValue('sources')
| :ref:`sourceFills <gridfile.fillValue.output.sourceFills>` = obj.fillValue('sources', :ref:`s <gridfile.fillValue.input.s>`)
| :ref:`sourceFills <gridfile.fillValue.output.sourceFills>` = obj.fillValue('sources', sourceNames)

.. raw:: html

    </pre>

Return the fill values for the specified data sources. If no sources are specified, returns the fill value for all data sources in the gridfile.


.. raw:: html

    <pre>

.. _gridfile.fillValue.syntax3:

.. rst-class:: syntax

| obj.fillValue(:ref:`fill <gridfile.fillValue.input.fill>`)

.. raw:: html

    </pre>

Set the default fill value for data loaded from a .grid file. Numeric data matching the fill value are converted to NaN when loading. This syntax sets the fill value for all data sources currently in the .grid file and applies the fill value by default to all data sources added in the future.


.. raw:: html

    <pre>

.. _gridfile.fillValue.syntax4:

.. rst-class:: syntax

| obj.fillValue(:ref:`fill <gridfile.fillValue.input.fill>`, :ref:`s <gridfile.fillValue.input.s>`)
| obj.fillValue(:ref:`fill <gridfile.fillValue.input.fill>`, sourceNames)

.. raw:: html

    </pre>

Implements a fill value for the specified data sources. The fill value overrides any fill values previously applied to the data sources.
