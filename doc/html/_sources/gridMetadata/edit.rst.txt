gridMetadata.edit
=================
Edit the metadata for a gridded dataset

----

Syntax
------

.. raw:: html

    <pre>

.. rst-class:: syntax

| :ref:`obj = obj.edit(dimension1, metadata1, dimension2, metadata2, .., dimensionN, metadataN) <gridMetadata.edit.syntax1>`
| :ref:`obj = obj.edit(..., 'attributes', attributes) <gridMetadata.edit.syntax2>`

.. raw:: html

    </pre>

----

Description
-----------

.. raw:: html

    <pre>

.. _gridMetadata.edit.syntax1:

.. rst-class:: syntax

| :ref:`obj <gridMetadata.edit.output.obj>` = obj.edit(dimension1, metadata1, dimension2, metadata2, .., :ref:`dimensionN <gridMetadata.edit.input.dimensionN>`, :ref:`metadataN <gridMetadata.edit.input.metadataN>`)

.. raw:: html

    </pre>

Replace the metadata for the named dimensions with the specified values.

If the gridMetadata has a set dimension order, and the defined dimensions change, also removes the dimension order from the object.


.. raw:: html

    <pre>

.. _gridMetadata.edit.syntax2:

.. rst-class:: syntax

| :ref:`obj <gridMetadata.edit.output.obj>` = obj.edit(..., 'attributes', :ref:`attributes <gridMetadata.edit.input.attributes>`)

.. raw:: html

    </pre>

Replace the non-dimensional attributes with the specified values
