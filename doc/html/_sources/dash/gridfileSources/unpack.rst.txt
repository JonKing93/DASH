dash.gridfileSources.unpack
===========================
Convert catalogued values to their original data types

----

Syntax
------

.. raw:: html

    <pre>

.. rst-class:: syntax

| :ref:`[dims, size, mergedDims, mergedSize, mergeMap] = obj.unpack(s) <dash.gridfileSources.unpack.syntax1>`

.. raw:: html

    </pre>

----

Description
-----------

.. raw:: html

    <pre>

.. _dash.gridfileSources.unpack.syntax1:

.. rst-class:: syntax

| [:ref:`dims <dash.gridfileSources.unpack.output.dims>`, :ref:`size <dash.gridfileSources.unpack.output.size>`, :ref:`mergedDims <dash.gridfileSources.unpack.output.mergedDims>`, :ref:`mergedSize <dash.gridfileSources.unpack.output.mergedSize>`, :ref:`mergeMap <dash.gridfileSources.unpack.output.mergeMap>`] = obj.unpack(:ref:`s <dash.gridfileSources.unpack.input.s>`)

.. raw:: html

    </pre>

Returns recorded values for a data source in the catalogue. Converts recorded values from optimized save/load data types to original data types.
