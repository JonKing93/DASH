dash.gridfileSources.absolutePaths
==================================
Return the absolute paths to data sources in the catalogue

----

------

| :ref:`paths = obj.absolutePaths <dash.gridfileSources.absolutePaths.syntax1>`

----

Description
-----------

.. raw:: html

    <pre>

.. _dash.gridfileSources.absolutePaths.syntax1:

.. rst-class:: syntax

| :ref:`paths <dash.gridfileSources.absolutePaths.output.paths>` = obj.absolutePaths

.. raw:: html

    </pre>

Returns the absolute paths to all data sources. Paths are returned in the order that sources are stored in the object.


.. raw:: html

    <pre>

.. _dash.gridfileSources.absolutePaths.syntax2:

.. rst-class:: syntax

| :ref:`paths <dash.gridfileSources.absolutePaths.output.paths>` = obj.absolutePaths(:ref:`s <dash.gridfileSources.absolutePaths.input.s>`)

.. raw:: html

    </pre>

Returns the absolute paths to the specified data sources. Paths are returned in the order of input indices.
