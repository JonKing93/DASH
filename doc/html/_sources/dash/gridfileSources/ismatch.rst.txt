dash.gridfileSources.ismatch
============================
Test if a dataSource object matches an entry in the catalogue

----

Syntax
------

.. raw:: html

    <pre>

.. rst-class:: syntax

| :ref:`tf = obj.ismatch(dataSource, s) <dash.gridfileSources.ismatch.syntax1>`
| :ref:`[tf, property, objValue, catalogueValue] = obj.ismatch(dataSource, s) <dash.gridfileSources.ismatch.syntax2>`

.. raw:: html

    </pre>

----

Description
-----------

.. raw:: html

    <pre>

.. _dash.gridfileSources.ismatch.syntax1:

.. rst-class:: syntax

| :ref:`tf <dash.gridfileSources.ismatch.output.tf>` = obj.ismatch(:ref:`dataSource <dash.gridfileSources.ismatch.input.dataSource>`, :ref:`s <dash.gridfileSources.ismatch.input.s>`)

.. raw:: html

    </pre>

Returns true if a dataSource object matches the specified source in the catalogue. Otherwise, returns false.


.. raw:: html

    <pre>

.. _dash.gridfileSources.ismatch.syntax2:

.. rst-class:: syntax

| [:ref:`tf <dash.gridfileSources.ismatch.output.tf>`, :ref:`property <dash.gridfileSources.ismatch.output.property>`, :ref:`objValue <dash.gridfileSources.ismatch.output.objValue>`, :ref:`catalogueValue <dash.gridfileSources.ismatch.output.catalogueValue>`] = obj.ismatch(:ref:`dataSource <dash.gridfileSources.ismatch.input.dataSource>`, :ref:`s <dash.gridfileSources.ismatch.input.s>`)

.. raw:: html

    </pre>

If the dataSource is not a match, returns information about the properties that do not match the values in the catalogue. If a match, the additional outputs are empty.
