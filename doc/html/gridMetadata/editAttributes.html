gridMetadata.editAttributes

Replace non-dimensional attributes with new values

Syntax



obj = obj.editAttributes(attribute1, value1, attribute2, value2, .., attributeN, valueN)

Description



obj = obj.editAttributes(attribute1, value1, attribute2, value2, .., attributeN, valueN)

Replaces the values of the named attributes with new values.

Input Arguments

attributeN

string scalar
The name of a field in the current gridMetadata object’s attributes structure.

valueN

The new value to use for the attribute.

Output Arguments

obj

gridMetadata object
The updated gridMetadata object
